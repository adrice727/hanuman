### Hanuman : A small, functional, JavaScript helper library


###### Hanuman provides a *minimal* set of utility functions:
* `curry`
* `forEach`
* `map`
* `filter`
* `reduce`
* `path`
* `pick`
* `pickAll`

###### **~ All functions are automatically curried, and accordingly, accept the data as the final argument ~**
